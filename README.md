# Tektronix AFG31000 Waveform Converter

Convert a csv file into AFG31000 csv format
![App](Assets/App.png)

## Input File format

***Decimal or Scientific notation***:arrow_forward: Can be Mixed types

### Decimal

```c
// Time(Seconds), Voltage(V)
0.0005, 5
0.0005, 0
0.0004, 4
0.0006, 0
0.0006, 3
0.0004, 0
0.0003, 3
0.0007, 0
```

### Scientific notation

```c
// Time(Seconds), Voltage(V)
5e-4, 5
5e-4, 0
4e-4, 4
6e-4, 0
6e-4, 3
4e-4, 0
3e-4, 3
7e-4, 0
```
